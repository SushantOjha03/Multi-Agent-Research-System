# Multi-Agent Research System

A Python-based multi-agent research system that automates research workflows using multiple AI agents. The system coordinates specialized agents to gather, process, and organize information efficiently, making it suitable for research automation and AI-assisted knowledge discovery.

## Features

* 🤖 Multi-agent architecture
* 🔍 Automated research pipeline
* 🛠 Modular and extensible design
* 📄 Easy-to-understand Python implementation
* ⚡ Lightweight project structure
* 🔄 Configurable agent workflow

## Project Structure

```text
Multi-agent-research-system/
│── agents.py           # Agent definitions and orchestration
│── app.py              # Application entry point
│── pipeline.py         # Research workflow pipeline
│── tools.py            # Utility functions and external tools
│── requirements.txt    # Python dependencies
│── .gitignore
```

## Installation

### 1. Clone the repository

```bash
git clone https://github.com/SushantOjha03/Multi-Agent-Research-System.git
cd Multi-Agent-Research-System
```

### 2. Create a virtual environment

Windows

```bash
python -m venv .venv
.venv\Scripts\activate
```

Linux / macOS

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

## Usage

Run the application:

```bash
python app.py
```
